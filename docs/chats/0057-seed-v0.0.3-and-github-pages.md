## Decisions / notes

- **v0.0.3-seed** is required for kernels that depend on literals, multi-clause matching, or prelude **`define-vibe-syntax`** for **`vibe:*`**; **v0.0.2-seed** remains for older macro-only trees.
- **GitHub Pages** stays a short public summary; **`macro-system.md`** is canonical for expander detail.
